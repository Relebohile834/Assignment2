# Assignment2

ST10508029 Relebohile Thato Phalatsi

Life Hacks and internet rumours are everywhere. Users often struggle to distinguish between genuine productivity tips and urban myths that have gone viral. I've created a native Android app using Kotlin within Android Studio to address this. This app provides a "Life Hack or Urban Myth" flashcard game to help users test their common sense and learn useful and safe real-world shortcuts. This app helps users study and verify life hacks with flashcards. The app presents questions, loops through each hack, and provides a total score based on the user's ability to spot a myth. In the app's user interface, there is a welcome screen or activity, a quiz activity, a score activity, and a review activity. In the welcome screen, a brief description of the app and a welcome message are present, with a start button to begin the quiz. I've also added images and made the colour scheme of the app interface appealing. In the quiz activity, a specific hack statement and two buttons for answer options (hack and myth) are present with a next button to load the next question after each answer. A feedback message for each answer is included to notify the user if they are correct or incorrect in distinguishing the myth or hack. An explanation is also included in the feedback message to further explain to the user why the statement is a hack or a myth. In the score activity, a feedback message appears depending on the user's results on whether they were able to distinguish a hack from a myth, and the total number of correct answers is provided. Personalized feedback is included based on the score that the user obtained. A review button is also present in the score activity to let users see the list of statements and their correct explanations in the review activity. I've made the user interface design to be appropriate and fun by making it consistent with each page, making it usable and visually appealing.
